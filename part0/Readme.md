# Parte 0 - Fundamentos de Aplicaciones Web

Esta carpeta contiene mis soluciones para la Parte 0 del curso Full Stack Open.

## Ejercicios

### 0.4 - Nueva Nota

Diagrama de secuencia que describe cómo se crea una nueva nota en una aplicación web tradicional.

### 0.5 - Single Page App (SPA)

Diagrama que muestra cómo funciona la versión SPA de la aplicación de notas.

### 0.6 - Nueva Nota en SPA

Diagrama que ilustra cómo se crea una nueva nota en la versión SPA:

- Los datos se envían en formato JSON
- Se utiliza una solicitud HTTP POST
- No hay recarga de página
- El servidor responde con estado 201 Created

## Notas

Estos ejercicios se enfocan en comprender:

- El ciclo request-response de HTTP
- La diferencia entre aplicaciones tradicionales y SPA
- La interacción cliente-servidor
